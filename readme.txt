==================================================
                 CMS芒果系统
===================================================
项目名：  CMS芒果系统
版本号：   v0.1
开发框架：  RAILS
开发环境：  ruby 1.9.3
           rails 3.2.1

cms制作目的： 针对客户对网站的需求量大、沟通不及时、有小问题，自己不能修改，为了减少修改流程，
             网络中心决定出一套由rails写的【芒果CMS系统】，此系统可以有效的解决建站时间长，用户修改不及时等问题...
             
版本功能：
   1、  自定义页面
           只针对栏目下只有一篇文章时应用、 例【公司介绍】、【联系我们】、【活动页面】
   2、  文章功能
           栏目下建立文章列表， 添加文章时选择相应的栏目，就会出现在该【栏目页】下。
   3、  产品功能
           产品本身是不参与售出的， 是为商品提供实物，产品添加后可做产品展示，显示该产品的说细信息
   4、  招聘功能
           使用该系统可以发布招聘信息 , 页面例： 【人员招聘】
   5、  模版功能
           模版分为：  系统默认模版和用户自定义模版
   6、  支付宝支付功能;
           在系统设置--> 设置支付宝账户，  在商城购买商品后就可以使用支付宝付未款。
